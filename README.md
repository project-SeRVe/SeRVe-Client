# SeRVe-Client
Python script for Robot Client

# 세팅

linux 기준 client 세팅법.

## 실행 환경설정

```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Ollama 설치 및 사용

실제로 추론을 수행하려면 ollama가 설치되어 있고, 실행 중이어야 한다.

https://docs.ollama.com/quickstart

## Client 실행하는법

```
source .venv/bin/activate
streamlit run app.py
```
